Story number and name: 1 - Extend user profile detail with usage
Story points: 3
Brief description to see the story in action:
Navigate to any user’s profile page. The page displays the user’s most recent photo and the photo with the highest number of comments.

Story number and name: 5 - Deleting comments, photos, and users
Story points: 4
Brief description to see the story in action:
As a logged-in user, delete your own comments and photos using the delete buttons. From the user profile page, delete the user account after confirming the warning; all associated photos and comments are removed and the user is logged out.

Story number and name: 6 - Photo "like" votes
Story points: 4
Brief description to see the story in action:
On any photo page, click the Like button to like or unlike a photo.
The like count updates in real-time across all connected clients using sockets,
and photos are sorted by like count (descending) with timestamp as a tie-breaker.

Story number and name: 7 - Favorite list of photos for users
Story points: 6
Brief description to see the story in action:
Users can favorite a photo by clicking on the "favorite" button. To remove a favorite they need to navigate to /favorites. Click the Favorites button in the toolbar to do so. From here, users can unfavorite photos, view thumbnails of favorite photos, and open them in a modal viewer.
